# bnc

Frontend Client for Benjamin and Charlotte game.

Demo: https://bnc-client-psi.vercel.app

Mono-repo: https://github.com/jwc20/bnc-game

Docs: https://github.com/jwc20/bnc-docs

## Quick Start

```bash
docker compose -f docker-compose.yml up -d --build
```

Alternatively,

```bash
yarn install
yarn dev
```

Access the site at http://localhost:5173/ or http://0.0.0.0:5173/
